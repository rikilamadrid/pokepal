# PokéPal

A mobile-first, iOS-style Pokémon card companion app for kids: scan a card, tag
it, and browse your collection. Local-first, installable as a PWA, and packaged
for the App Store / Play Store via Capacitor.

Built with **Next.js 16 (App Router, static export) · React 19 · TypeScript ·
Tailwind CSS v4 · shadcn/ui**, backed by **Supabase** for accounts and cloud
sync.

## Getting started

```bash
npm install
npm run dev      # http://localhost:3000
```

```bash
npm run build    # static export → out/
npm run lint
```
